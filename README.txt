C195 Scheduling Project
The purpose of this project was to create a scheduling application to create customers and appointments and store it to a database.

Author: Sainikit Gorupati
Application Version: 1.1
Date Submitted: 12/14/2021

IDE: IntelliJ IDEA 2021.2.2 (Ultimate Edition)
JDK version: Java SE 11.0.12
JavaFX version: JavaFX SDK 17.0.1

To run program:
Run the main file through IntelliJ. Once the login screen pops up enter "test" into the username and password fields.
Then the application will open to main screen which will allow the application to be used.

Additional Report:
The additional report provided is Past Appointments. This report shows all the appointments in the database that have
already occurred.

MySQL Connector driver version: mysql-connector-java-8.0.27
